# LDRFlow

## Datasets
Build the `data` dir, and put the downloaded data in it. By default `datasets.py` will search for the datasets in the location.
```Shell
├── data
    ├── Raw
        ├── Test
        ├── Training
        ├── Training_ICCV
    ├── IMF_short2long
        ├── Test
        ├── Training
        ├── Training_ICCV
    ├── IMF_table
        ├── Training
        ├── Training_ICCV
    ├── GT_flow
```
